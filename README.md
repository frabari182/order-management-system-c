# Order Management System (C)

University project developed individually in C.

The project simulates an order management system for a bakery with inventory handling,
priority-based scheduling, and discrete-time events.

## Technologies
- Language: C
- Data structures
- File I/O

## Main Features
- Modular program design
- Inventory and order management
- Priority-based scheduling
- Discrete-time simulation

## How to Compile
```bash
gcc src/main.c -o order_management
